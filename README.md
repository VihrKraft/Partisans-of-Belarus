# Партизаны в Беларуси в годы Великой Отечественной Войны

Сайт является познавательным ресурсом, который позволяет узнать основные события, связанные с Партизанами в годы ВОВ, и некольких известных партизанах.

[Сайт на GitHup Pages](https://vihrkraft.github.io/Partisans-of-Belarus/dist/html/home.html)

## Как скачать

Сайт можно запустить предварительно скачав файлы. Существует два способа: 
1. Скачать проект в виде архива (`Add file` => `Upload files`) и разархивировать его в нужном для вас месте.
2. Сделать `Fork` проекта и клонировать его на компьютер с помощью `GitHub Desktop`.

После проделанного зайти в корневую папку сайта в проводнике, зайти в папку `dist` => `html` и открыть любой интересующий вас файл (например, `home.html`) в браузере.

## Описание файлов

Весь код сайта находится в папке `dist`:
1) В папке `video` - видео, используемое на странице `videoPlayer.html`.
2) В папке `html` -  все файлы структуры сайта.
3) В папке `css` - файлы стилей соответственно названиям `.html` файлов.
4) В папке `img` - все используемые изображения.
5) В папке `js` - JavaScript файлы, делающие страницы сайта более плавными (`playable.bundle.min.js`, `player.js` и `jquery-3.4.1.min.js` нужны только для видеоплеера).
6) В папке `audio` - аудиофайл, используемый на `main.html`.
7) В папке `videoPlayerStyles` - набор стилей, нужный для корректного отображения [видеоплеера](https://vihrkraft.github.io/Partisans-of-Belarus/dist/html/videoPlayer.html).
